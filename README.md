# Muhammad Ammar — Personal Portfolio Landing Page

A clean, single-page personal portfolio website built with plain HTML and CSS. It introduces Ammar as a software engineer, highlights his technical skills, showcases sample project cards, and includes a contact section — all in one responsive landing page.

LIVE LINK : https://tcintern-006.github.io/Personal-Portfolio-Landing-Page/
## ✨ Features

- **Hero header** — name, title, a short tagline, profile photo, and a "Download CV" button
- **Social links** — GitHub, LinkedIn, and YouTube icons (Font Awesome)
- **About section** — short bio and a link to open LinkedIn
- **Skills section** — animated Font Awesome icons for the stack: HTML5, CSS3, JavaScript, React, Node.js, Express.js, MongoDB, JWT Authentication, REST APIs, Git, GitHub, npm, command line, responsive design, full-stack development, and cloud deployment
- **Projects section** — card layout showcasing sample MERN-stack projects (e-commerce site, hospital management system, portfolio site, task manager, real-time chat app) with placeholders for live demo and GitHub links
- **Contact section** — a simple contact form (name, email, subject, message)
- **Footer** — copyright notice

## 🛠️ Built With

- **HTML5** — page structure and content
- **CSS3** — styling and layout (`style.css`)
- **[Font Awesome](https://fontawesome.com/)** — icon set, loaded via CDN

## 🚀 Getting Started

No build tools or dependencies required — this is a static site.

1. Clone the repository
   ```bash
   git clone https://github.com/tcintern-006/Personal-Portfolio-Landing-Page.git
   ```
2. Open the project folder
   ```bash
   cd Personal-Portfolio-Landing-Page
   ```
3. Open `index.html` directly in your browser, or serve it locally (e.g. with the VS Code Live Server extension) for the best experience.
